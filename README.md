Descrição do projeto

- Projeto voltado ao desenvolvimento da estrutura de automação de testes para a aplicação Urban Routes. Foram configurados o ambiente de testes, a organização do projeto e os casos de teste automatizados utilizando Python, Selenium WebDriver e PyTest, seguindo boas práticas de programação e automação.

1. Objetivo do projeto

- Desenvolver uma base para automação de testes funcionais da aplicação, estruturando o projeto para validar os principais fluxos do sistema de forma automatizada.

2. Resultado

- Foi criada a estrutura inicial do projeto de automação, com organização modular do código, definição dos cenários de teste e configuração do ambiente utilizando Selenium e PyTest. Também foram implementadas funções auxiliares e dados reutilizáveis para facilitar a manutenção dos testes.

3. Ferramentas utilizadas

- Python;
- Selenium WebDriver;
- PyTest;
- Git e GitHub;
- Visual Studio Code;
- Page Object (conceitos);
- Automação de Testes Web.
5. O que aprendi nesse projeto (habilidades/competências adquiridas)

- Estruturação de projetos de automação de testes;
- Desenvolvimento de testes automatizados com Python;
- Utilização do Selenium WebDriver para interação com aplicações web;
- Organização modular do código e reutilização de componentes;
- Aplicação de boas práticas de automação utilizando PyTest.

6. Existem melhorias a serem feitas?

- Como evolução do projeto, seria interessante ampliar a cobertura dos testes automatizados, implementar o padrão Page Object Model (POM) de forma completa, adicionar relatórios de execução e integrar os testes a pipelines de Integração Contínua (CI/CD).

- Metodologia

- Configuração do ambiente de automação, organização da estrutura do projeto, implementação dos casos de teste utilizando Python e Selenium, execução dos testes com PyTest e validação dos resultados obtidos.

Como executar o projeto

- Instalar as dependências do projeto (requirements.txt), configurar o ambiente Python, iniciar a aplicação Urban Routes e executar os testes utilizando o PyTest para validar automaticamente os fluxos implementados.
